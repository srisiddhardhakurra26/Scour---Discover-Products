# Scour

Discover products across major shopping platforms and compare them side by side.

**Status:** early design phase. No usable code yet — see [`docs/`](docs/) for the full design.

## What it does (planned)

- Type a query → Scour searches Amazon, AliExpress, Temu, Shein, eBay, Walmart, Etsy, indie Shopify stores, etc. in parallel.
- Same product across retailers is clustered into one card with prices side by side.
- Add custom sources (Shopify storefronts), follow categories, set price-drop alerts.
- Image search: upload a photo, find visually similar products on every supported store.
